# Background Cropping Bot 🤖🎨

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Aiogram](https://img.shields.io/badge/aiogram-3.3.0-green.svg)](https://docs.aiogram.dev/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Telegram бот для автоматического удаления фона с фотографий и его замены с помощью нейросетей. Просто, быстро и удобно!


## ✨ Возможности

*   **Мгновенное удаление фона**: Отправьте фото и получите результат через несколько секунд.
*   **Замена фона**:
    *   На **белый** или **черный** цвет одним нажатием.
    *   На **любое другое изображение**, которое вы отправите.
*   **Статистика**: Отслеживайте количество обработанных вами фотографий.
*   **Прогресс-бар**: Бот показывает этапы обработки, чтобы вы не скучали в ожидании.
*   **Удобная навигация**: Все функции доступны через понятные кнопки меню.


## 🛠 Технологии

*   **Язык**: Python 3.8+
*   **Основные библиотеки**:
    *   `aiogram` — для работы с Telegram API.
    *   `rembg` — нейросеть для удаления фона.
    *   `Pillow` — для обработки изображений (наложение фона).
    *   `python-dotenv` — для безопасного хранения токена.

## 📋 Установка и запуск

Следуйте этим простым шагам, чтобы запустить свою копию бота.

### Предварительные требования

*   Установленный Python 3.8 или выше.
*   Токен вашего Telegram бота (получить у [@BotFather](https://t.me/botfather)).

### Пошаговая инструкция

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/flivergg/background-cropping-bot.git
    cd background-cropping-bot
Создайте и активируйте виртуальное окружение (рекомендуется)

Windows:

bash
python -m venv venv

venv\Scripts\activate

Linux/macOS:

bash
python3 -m venv venv

source venv/bin/activate

Установите зависимости

bash
pip install -r requirements.txt
Настройте переменные окружения

Создайте файл .env в корневой папке проекта.
Добавьте в него следующие строки, заменив значения на свои:

env
BOT_TOKEN=ВАШ_ТОКЕН_БОТА_ОТ_BOTFATHER

ADMIN_ID=ВАШ_TELEGRAM_ID

BOT_TOKEN: токен, который вы получили от BotFather.

ADMIN_ID: ваш числовой ID в Telegram (узнать можно у бота @userinfobot). Нужен для доступа к админ-командам.

Запустите бота

bash
python bot.py

Если вы видите в консоли Запускаем бота... — всё работает! 🎉
🎮 Как пользоваться

Найдите своего бота в Telegram и нажмите Start.
В главном меню нажмите кнопку «🎯 Удалить фон».
Отправьте фото.
Дождитесь окончания обработки. Бот пришлет фото с удаленным фоном.
Чтобы заменить фон, нажмите «🎨 Изменить фон» и выберите один из вариантов:

Цветной фон (белый или черный).
Свое фото — отправьте картинку, которая станет новым фоном.
👑 Команды администратора

Если вы указали свой ADMIN_ID в файле .env, вам доступна панель администратора по команде /admin. В ней можно:

Просмотреть общую статистику бота (количество пользователей, обработок).
Увидеть список последних пользователей.
