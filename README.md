🤖 AI Photo Editor Bot

Telegram бот для удаления фона с фотографий и замены его на любой цвет или другое изображение с использованием нейросетей.

✨ Возможности

🎯 Удаление фона - мгновенное удаление фона с любых фотографий
🎨 Замена фона - замена на белый, черный фон или любое другое фото
📊 Статистика - отслеживание количества обработанных фото
👑 Админ-панель - управление и статистика для администратора

📋 Команды

Команда	Описание
/start	Запуск бота
/admin	Вход в админ-панель
Кнопки меню:

🎯 Удалить фон - начать обработку фото
🎨 Изменить фон - заменить фон после удаления
📊 Статистика - ваша личная статистика
🆘 Помощь - инструкция по использованию
🛠 Установка

Требования

Python 3.8+
Telegram Bot Token (получить у @BotFather)
Пошаговая инструкция

Клонировать репозиторий
bash
git clone https://github.com/yourusername/photo-editor-bot.git
cd photo-editor-bot
Создать виртуальное окружение
bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
Установить зависимости
bash

pip install -r requirements.txt
Настроить переменные окружения
bash

# Создайте файл .env и добавьте:
BOT_TOKEN=ваш_токен_от_botfather
ADMIN_ID=ваш_telegram_id
Запустить бота
bash
python main.py
